# Pigeon-on-Steroids
Swift Pigeon is a real-time messaging application that enables users to communicate instantly over the internet. The application allows users to log in, send and receive messages, view chat history, and securely log out.

The system uses a modern frontend technology stack with Firebase as the backend, providing real-time message synchronization and secure authentication.

This project was developed as part of the Software Engineering and Agile Development Lab, following Agile and Extreme Programming (XP) practices.

**Features**

**User Features**
Secure user authentication (Login)
Real-time message sending
Instant message receiving
Chat history viewing
Secure logout

**Admin Features**
Manage user accounts
Monitor system activity
Future Enhancements
Online/offline user status
Message read receipts
File and image sharing
Group chats
Push notifications
